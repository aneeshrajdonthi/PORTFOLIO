Comicify

This Python tool converts YouTube videos into a comic-style PDF, extracting key frames and adding dialogues for an engaging visual storytelling experience.

Domino-s-Pizza-Sales---Power-Bi-Dashboard

This Power BI dashboard provides insights into Domino’s Pizza store sales for 2023. By analyzing sales trends, customer preferences, and performance metrics, it helps optimize business strategies for growth and efficiency.

Key Metrics

  • Total Revenue & Orders
  • Total Pizzas Sold
  • Average Pizzas Per Order
  • Average Order Value

  • Peak Sales Days: Highest orders on weekends and Mondays
  • Monthly Trends: Maximum sales from January to July

Banking System

The Bank Data Extraction and Transformation Project is an ETL (Extract, Transform, Load) pipeline designed to gather and process financial data of the largest banks worldwide. Using web scraping, it extracts bank names and market capitalization (MC) from a Wikipedia page. The extracted data is then transformed by converting MC values from USD to GBP, INR, and EUR using exchange rates. Finally, the processed data is stored in a CSV file or database for further analysis.

Key Features

  • Web Scraping: Extracts bank names and market capitalization (MC) from Wikipedia.
  • Data Transformation – Converts market capitalization values into multiple currencies.
  • Storage Options – Saves the cleaned and transformed data in a CSV file and database.
  • Logging System – Maintains logs of operations for tracking and debugging. This project streamlines financial data processing and enables easy analysis of global banking trends.

Tools Used: Python, BeautifulSoup & Requests, MySQL/SQLite, Pandas, Excel, Logging Module
